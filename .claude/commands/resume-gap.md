분석 기간은 기본 7일입니다. 다른 기간을 원하면 숫자를 명시하세요 (예: "14일치 분석해줘").

다음 순서로 실행하세요:

**1단계 — 데이터 수집 및 키워드 집계**

```bash
python3 resume_gap.py --days 7
```

출력된 JSON 결과를 읽고 아래를 분석하세요:
- `gaps`: 이력서에 없는데 공고에 자주 나오는 기술
- `have`: 이미 이력서에 있고 공고에도 많이 나오는 강점

**2단계 — 이력서와 함께 해석**

`resume/이력서.txt`와 `resume/경력기술서.txt`를 읽고 다음을 판단하세요:

1. gaps 중 **실제로 이력서에 추가할 만한 것** vs **현재 경력과 무관한 것** 구분
2. gaps 중 **학습 우선순위** (취업에 가장 직접적인 영향을 줄 것)
3. have 중 **더 강조할 수 있는 경험** (공고 JD 키워드와 이력서 내용 연결)
4. 전체 트렌드에서 **지원 전략적 조언** 1~2줄

**3단계 — Discord 발송**

인사이트를 한 단락으로 요약한 뒤 아래 커맨드로 발송하세요:

```bash
python3 resume_gap.py --send --days 7 --insight "여기에 인사이트 요약 텍스트"
```
